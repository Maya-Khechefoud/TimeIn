# TimeIn

TimeIn is a mobile-first planner built through **vibe coding**.

After trying many planner and to-do list applications, I couldn't find one that matched both the design I wanted and the features I needed. Instead of adapting my workflow to existing tools, I decided to build my own.

TimeIn combines scheduling and progress tracking in a simple, clean, and intuitive interface designed primarily for mobile devices.

## Features

### 📅 Schedule

The Schedule page is the main workspace of TimeIn.

Users can:

* Create tasks
* Edit tasks
* Delete tasks
* Mark tasks as completed
* Organize tasks by category

Available categories:

* Personal
* Business
* Study
* Shopping

Each task contains a title, description, time slot, and completion status, making it easy to organize daily activities and stay focused.

### 📊 Track

The Track page is dedicated to progress visualization.

I believe that seeing proof of progress and hard work is one of the best ways to stay consistent and motivated. That's why TimeIn includes a battery-style tracker that visually represents productivity.

The tracker provides:

* Completion percentage
* Number of completed tasks
* Number of remaining tasks
* Daily overview
* Weekly overview (not developped yet just interface)
* Monthly overview (not developped yet just interface)

The battery fills according to completed tasks, giving users an instant visual representation of their progress.

## Vision

TimeIn is designed to be a personal productivity tool that focuses on simplicity, consistency, and motivation.

The goal is not to create another complex planner, but to build a tool that feels natural to use every day while providing clear feedback on progress.
